# PlacementIQ

PlacementIQ is a multi-tenant, AI-powered Placement Risk Modeling platform designed for education loan lenders, NBFCs, and banks. It predicts how quickly a student will secure employment after graduation, estimates their likely starting salary, and continuously surfaces early-warning signals for borrowers at risk of delayed placement.

## Architecture & Folders
- **`backend/`**: FastAPI application serving predictions and powering the dashboards.
- **`frontend/`**: Next.js application for the Lender Dashboard and Student Career Co-Pilot.
- **`ml_engine/`**: Machine Learning pipelines, models (XGBoost, LightGBM), and evaluation scripts.
- **`infra/`**: DevOps, deployment configurations, and local Docker compose files.
